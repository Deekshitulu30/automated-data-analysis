# 📊 Automated Data Analysis Tool (AutoEDA)

## 📌 Project Overview

The **Automated Data Analysis Tool** is a Python-based web application that allows users to upload a dataset and automatically generate insights.
It helps perform **Exploratory Data Analysis (EDA)** without writing any code.

The tool provides statistical summaries, correlation analysis, and visualizations for better understanding of the dataset.

---

## 🎯 Objectives

* Simplify data analysis for beginners
* Automatically generate dataset insights
* Visualize relationships between variables
* Provide interactive charts for better understanding

---

## 🚀 Features

* Upload CSV dataset
* View dataset preview
* Automatic statistical summary
* Detect missing values
* Correlation heatmap
* Histogram visualization
* Scatter plot visualization
* Interactive column selection

---

## 🛠 Technologies Used

* **Python**
* **Pandas** – Data manipulation and analysis
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical visualization
* **Streamlit** – Web application interface

## ⚙ Installation

### 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/auto-data-analyzer.git
```

### 2️⃣ Navigate to Project Folder

```
cd auto-data-analyzer
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

## ▶ Running the Application

Run the following command:

```
python -m streamlit run automated_data_analysis.py
```

After running, open the browser and go to:

```
http://localhost:8501
```

---

## 📊 How It Works

1. Upload a CSV dataset.
2. The system reads the dataset using Pandas.
3. Statistical analysis is generated automatically.
4. Visualizations such as histograms and scatter plots are created.
5. A correlation heatmap shows relationships between numerical variables.
